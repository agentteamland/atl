# docs — the in-repo durable-knowledge store (GitHub backend)

The GitHub-backend delivery-team's durable-knowledge store (adapter §9). The ceremonies
seed and read namespaced Markdown here — `docs/domain/`, `docs/analysis/`,
`docs/architecture/`, `docs/conventions/`, `docs/sprints/` — instead of an Azure project
wiki. This baseline ships only this README; the ceremonies create the namespace pages.

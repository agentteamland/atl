# teams/

First-party ATL teams live here as monorepo subpaths (each with a `team.json` carrying a `scope` declaration — `global | project | both` — per the v2 scope axis).

**Status: empty by design (2026-07).** The v1-era teams (software-project-team, design-system-team) were retired — they were v1 ports never brought up to the v2 bar, and a deliberate rebuild on the v2 foundation beats patching them. Their history is preserved in git. The rebuild starts with **profile-team**, followed by a new software developer team.

# personal-advisory-team

A **global-scope** first-party ATL team — the head of the personal advisory stack.
It gives you a single honest, wise **primary advisor** that comes to know you across
every conversation and, because it won't flatter you, becomes the one you run to for
real thinking.

- **Install:** `atl install agentteamland/personal-advisory-team`
- **Depends on:** `profile-team` (pulled in transitively) — the shared, cross-project
  profile the advisor uses to actually know you.
- **How to use it:** `/advisor` starts (or resumes) a conversation; `/advisor-home`
  sets up an always-on advisory home + an `advisor` shell command you can open from
  any terminal.

Full documentation lives on the docs site: <https://docs.agentteamland.com/>.
